3. 创建密码，点击 Register；
4. 选择 No,thanks，点击 Continue；
5. 网页下拉至底部选择 Pay as you go，点击 Continue；
6. 点击 Or redeem a top-up voucher；
7. 填写姓名和地址（可以打开谷歌地图搜索英国地址），Voucher code 处输入充值券（没有的话联系顶部链接里客服），勾选 I understand and agree， 点击 Activate my SIM。
8. 点击左上角 logo 出现号码和余额，插卡等待有信号即可。
